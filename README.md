
## Run in codespaces or local

For local execution, additionally set up python and node. 

Setup backend

```bash
    cd chapter_x/
    cd frontend/
    pip install requirements.txt
```


Start the backend

```bash
  cd chapter_x/
  cd backend/
  fastapi dev main.py
```

Setup frontend

```bash
    cd chapter_x/
    cd frontend/
    npm install
```

Start frontend server

```bash
  npm run start
```
